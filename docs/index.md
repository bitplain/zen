# Get started

For full documentation visit [zensical.org](https://zensical.org/docs/).

> **Ready to build?** Kick off a site with `zensical new`, add pages, and ship with `zensical build`.

## Start here

- [Getting started](how-to/getting-started.md) — the quickest path from scaffold to published site.
- [CLI commands](platform/commands.md) — the essential commands you will run every day.

## Key sections

- **Processes:** [Content workflow](processes/content-workflow.md) for drafting, reviewing, and publishing updates.
- **How-to guides:** Recipes and step-by-step guides for new contributors.
- **Reference:** [Authoring examples](reference/authoring-examples.md) and a [Markdown primer](reference/markdown.md) you can paste into pages.
- **FAQ:** Common answers to keep teams unblocked. See [FAQ](faq/index.md).

Store shared visuals in `docs/assets/` and keep editable diagram sources in [`docs/assets/diagram-sources`](assets/diagram-sources/README.md) for easy updates.
