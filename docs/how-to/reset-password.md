# Сброс пароля

!!! info "Когда использовать"
    Внутренние запросы пользователей на восстановление доступа.

## Предпосылки

- Роль администратора в панели управления
- Доступ к сервис-деску для логирования кейса

## Шаги

1. Откройте панель администратора → Users.
2. Найдите пользователя по email и нажмите **Reset password**.
3. Подтвердите действие и отправьте одноразовую ссылку пользователю.
4. Создайте комментарий в тикете с временем выдачи ссылки.

??? info "Если нет доступа"
    Обратитесь в SecOps — они могут выдать временный токен или обновить роль.

## Проверка результата

- Пользователь подтвердил вход
- В тикете зафиксирована ссылка и время
